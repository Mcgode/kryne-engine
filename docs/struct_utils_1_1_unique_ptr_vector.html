Erases a unique pointer from the vector.

Will erase the unique pointer by left shifting the unique pointers values (using swap()) until reaching the end of the iterator, upon which the last element will be popped.

Inserts a unique pointer to the vector.

Will insert the unique pointer by right shifting the unique pointers values (using swap()) until reaching the end of the iterator, upon which a new unique pointer will be emplaced with the last value.

Complexity is linear.

Parameters
positionThe iterator pointing to the insert position.
valueThe unique pointer to insert into the vector.
Returns
An iterator pointing to the newly inserted element.
