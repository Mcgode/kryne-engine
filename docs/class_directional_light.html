◆ setCascadedShadowMaps()

void DirectionalLight::setCascadedShadowMaps ( const uint8 &  value)
inline

Updates the amount of cascaded shadow maps for this light.

The engine supports up to 4 cascaded shadow maps at this time, so the value will automatically be clamped between 1 and 4.

For cascading, the camera frustum depth is divided into 2^n-1 segments, with individual shadow maps handling 2^i segments. It means that each shadow map handles half the depth of the next shadow map.
