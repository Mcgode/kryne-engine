◆ computeBoundingVolumes()

void BufferGeometry::computeBoundingVolumes ( )

Computes the bounding sphere for this specific geometry, based on the the 'position' attribute.

Is automatically called when the 'position' attribute is set (using setAttribute()) or removed (using removeAttribute()).
Be sure to call this function if you change the position attribute data in-place.
