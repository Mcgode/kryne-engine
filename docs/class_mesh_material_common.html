Detailed Description

A child class of Material containing the common values used by most materials.

Note
If deriving from this class, don't forget to call initializeCommon() in the child class constructor

Initializes a material by attaching a shader to it.

Parameters
shaderThe shader for this material.

Calls Material::Material(unique_ptr<Shader>) directly. The call to initializeCommon() must be done manually, to allow for more initialization flexibility of children.
