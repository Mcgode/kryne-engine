◆ unsafeSetPreprocessingRequirement()

void Entity::unsafeSetPreprocessingRequirement ( Entity *  e)
inline

Sets an entity that needs to be preprocessed before preprocessing this entity in an unsafe way.

This property will ensure that the provided entity will be pre-processed before the current entity.
To be used in moderation, as it might increase execution time or lead to deadlock situations.

Warning
You need to update this manually, and reset the value when needed, to avoid segmentation faults.
You should also be very careful with its use. If not, you might create a dependency cycle and get locked in an infinite loop.
Parameters
eThe entity that needs to have already been preprocessed.
